## Status

Minnesota has the strongest official-source foundation and a broad existing inventory. Remaining work should be selective rather than cadence-driven.

## Main Blockers

- Candidate saturation on already-covered corridors.
- Some northern or secondary routes have weak route-specific gauge/scoring support despite public access.
- Remote routes may be better as overnight or advanced planning content than simple day-route additions.

## Current Guidance

- Prefer Minnesota DNR interpreted river-level gauges and state water trail maps.
- Add routes only where they materially improve coverage: new corridor, missing public-water-trail segment, or a distinct high-use access pair.
- Avoid duplicating dense corridors unless evidence and user value are clearly stronger than existing routes.
