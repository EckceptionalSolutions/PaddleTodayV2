## Main Blockers

- Weak numeric thresholds on popular flatwater/water-trail corridors.
- No live gauge on many Upper Peninsula and smaller whitewater leads.
- Gauge proxy weakness where a nearby river or downstream dammed gauge does not safely represent the route.
- Temporary closures: Huron River Barton/Argo corridor, Rogue River Grand Rogue launches, lower Boardman downtown work, Grand Rapids whitewater project.
- Some AW whitewater/play features are rejected or deferred because they are not normal PaddleToday route records.
